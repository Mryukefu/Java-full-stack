# 常见java类
  ## 八大基础类型
   ### byte 字节类型
   * 数字范围<br>
   一个字节存储8位无符号数,储存的数值范围为255个数字2^7-1--2^7:(-128-127)
   * 应用场景<br>
   字节流存储,因为在基本数据类型中是最小的单位，所以都可以向上转型
   
   ### short 浮点类型
   * 数字范围<br>
    一个浮点类型16位无符号数,储存的数值范围为255个数字2^16-1--2^16
    精度可以到小数点六位小数
   * 应用场景<br>
    精度要求不高的数值，除不尽时候容易死循环
